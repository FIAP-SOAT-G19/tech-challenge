# "Deleta" o cadastro de um cliente

> ## Caso de sucesso

1. ⛔ Atualiza o campo deletedAt do cadastro de um cliente

1. ⛔ Retorna 400 se o id do cliente não for fornecido
2. ⛔ Retorna 400 se o id informado for inválido
